# Курсовой проект по дисциплине «ТРПО»

## Тема РГР

**Игра «Жизнь» (Conway's Game of Life)**

## Описание проекта

Проект представляет собой консольное приложение на языке C, моделирующее клеточный автомат «Игра Жизнь». Программа умеет:

- читать начальное состояние поля из файла или стандартного ввода;
- выполнять пошаговое моделирование поколений;
- запускать интерактивную анимацию в терминале;
- управлять скоростью моделирования во время выполнения.

Проект подготовлен с учётом критериев курсовой работы: законченный продукт, система сборки, модульные тесты, и CI.

## Структура репозитория

```text
GameOfLife
|- .github/workflows/ci.yml   # автоматическая сборка и тесты
|- docs/                      # ТЗ

|- src/                       # исходный код приложения
|- tests/                     # unit-тесты

| README.md

```
